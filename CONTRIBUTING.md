## ğŸ¤ Cara Berkontribusi

### 1. Fork Repository

```bash
# Fork via GitHub UI, kemudian clone
git clone https://github.com/YOUR-USERNAME/github-project-example.git
cd github-project-example
```

### 2. Buat Branch

```bash
git checkout -b feature/nama-fitur
# atau
git checkout -b fix/nama-bug
# atau
git checkout -b docs/nama-dokumen
```

### 3. Buat Perubahan

Ikuti panduan style:

- Gunakan Markdown untuk dokumentasi
- Sertakan contoh yang jelas
- Tambahkan screenshot jika membantu
- Tulis dalam Bahasa Indonesia yang baik

### 4. Commit

```bash
git add .
git commit -m "feat: menambahkan latihan advanced filtering"
```

**Commit Message Format:**

- `feat:` untuk fitur baru
- `fix:` untuk bug fix
- `docs:` untuk dokumentasi
- `style:` untuk formatting
- `refactor:` untuk refactoring
- `test:` untuk testing
- `chore:` untuk maintenance

### 5. Push & Pull Request

```bash
git push origin feature/nama-fitur
```

Buat Pull Request di GitHub dengan:

- Title yang jelas
- Deskripsi perubahan
- Screenshot jika UI changes
- Link ke issue terkait
